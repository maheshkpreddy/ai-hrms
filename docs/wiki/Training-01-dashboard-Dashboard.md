## 📋 Module Overview

The Dashboard is the central command center of eh2r AI, providing a role-based overview of key HR metrics, KPIs, and quick actions. It adapts to each user's role, displaying relevant information such as employee headcount, attendance summaries, leave calendars, and recent activity feeds. The dashboard enables administrators to monitor company-wide statistics and trends, managers to track team performance and pending approvals, and employees to view personal summaries and quick actions.

The Dashboard module is accessible from both the sidebar navigation and the module home page grid. It contains the following sub-items that provide specialized functionality within the module.

---

## 🔧 Sub-Items & Features

### 1. Overview

Displays key performance indicators including total employees, active employees, new hires, and department breakdown. The overview section provides at-a-glance metrics with visual charts and trend indicators to help you understand the current state of your organization.

**Steps to use Overview:**
1. Navigate to the Dashboard module from the sidebar or module home page
2. Click on "Overview" from the sub-menu
3. Review the displayed metrics and charts for insights
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

### 2. Quick Actions

One-click access to common tasks such as adding a new employee, approving leave requests, running payroll, and scheduling meetings. Quick actions are context-aware based on your role.

**Steps to use Quick Actions:**
1. Navigate to the Dashboard module from the sidebar or module home page
2. Click on "Quick Actions" from the sub-menu
3. Use the available options to create, view, edit, or manage records
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

### 3. Recent Activity

A real-time feed showing the latest activities across the system including new employee additions, leave approvals, payroll processing, and document uploads. Each activity entry includes a timestamp and the user who performed the action.

**Steps to use Recent Activity:**
1. Navigate to the Dashboard module from the sidebar or module home page
2. Click on "Recent Activity" from the sub-menu
3. Use the available options to create, view, edit, or manage records
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

## 👤 Role Access

The following roles have access to the Dashboard module:

- **Super Admin**
- **HR Admin**
- **Department Manager**
- **Employee**
- **Payroll Specialist**
- **Recruiter**
- **L&D Manager**

> 💡 **Note**: The level of access (read, write, modify, delete) varies by role. Super Admins have full access, while other roles may have restricted permissions. Check the [Role Access Matrix](https://github.com/maheshkpreddy/ai-hrms/wiki/Home#role--module-access-matrix) for details.

## 📝 Step-by-Step Usage Instructions

### Getting Started
1. **Log in** to eh2r AI at [https://ai-hrms-rho.vercel.app](https://ai-hrms-rho.vercel.app) with your credentials
2. **Navigate** to the Dashboard module using the sidebar or module home page
3. **Select** the appropriate sub-item from the module menu
4. **Perform** the desired action (create, view, edit, or manage records)
5. **Save** your changes and verify the results

### Common Actions
- **Overview**: Access via Dashboard → Overview
- **Quick Actions**: Access via Dashboard → Quick Actions
- **Recent Activity**: Access via Dashboard → Recent Activity

### Best Practices for Dashboard
- Always review data before submitting or saving changes
- Use search and filter options to quickly find specific records
- Check for validation errors and correct them before saving
- Log out when you are done to protect your account security

---

## 💡 Tips & Best Practices

1. Use the dashboard as your starting point each day to stay updated on important metrics and pending actions.
2. Quick actions can save significant time — use them instead of navigating through menus for common tasks.
3. The dashboard content changes based on your role, so you only see what's relevant to you.
4. Check the recent activity feed regularly to stay informed about team updates and system changes.
5. Customize your dashboard view by pinning frequently used widgets and reports.
