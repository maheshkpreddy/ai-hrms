## 📋 Module Overview

The My Profile module provides a focused view of the current user's personal and employment information. Unlike the broader Employee Self-Service module, My Profile is dedicated to viewing and managing the user's own detailed profile including personal information, employment history, and associated documents. It serves as the single source of truth for an employee's data within the system and provides quick access to profile editing and document management.

The My Profile module is accessible from both the sidebar navigation and the module home page grid. It contains the following sub-items that provide specialized functionality within the module.

---

## 🔧 Sub-Items & Features

### 1. Personal Info

View and edit personal details including full name, date of birth, gender, contact information (phone, email), address, and emergency contacts. Some fields may require HR approval for changes.

**Steps to use Personal Info:**
1. Navigate to the My Profile module from the sidebar or module home page
2. Click on "Personal Info" from the sub-menu
3. Use the available options to create, view, edit, or manage records
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

### 2. Employment

View employment details including employee ID, department, designation, job title, contract type, join date, salary, and reporting manager. Track employment status and contract details.

**Steps to use Employment:**
1. Navigate to the My Profile module from the sidebar or module home page
2. Click on "Employment" from the sub-menu
3. Use the available options to create, view, edit, or manage records
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

### 3. Documents

Access and upload personal documents including ID proofs, certificates, contracts, and other employment-related documents. Manage document visibility and access permissions.

**Steps to use Documents:**
1. Navigate to the My Profile module from the sidebar or module home page
2. Click on "Documents" from the sub-menu
3. Use the available options to create, view, edit, or manage records
4. Apply filters or search as needed to find specific items
5. Save or submit your changes when complete

## 📝 Step-by-Step Usage Instructions

### Getting Started
1. **Log in** to eh2r AI at [https://ai-hrms-rho.vercel.app](https://ai-hrms-rho.vercel.app) with your credentials
2. **Navigate** to the My Profile module using the sidebar or module home page
3. **Select** the appropriate sub-item from the module menu
4. **Perform** the desired action (create, view, edit, or manage records)
5. **Save** your changes and verify the results

### Common Actions
- **Personal Info**: Access via My Profile → Personal Info
- **Employment**: Access via My Profile → Employment
- **Documents**: Access via My Profile → Documents

### Best Practices for My Profile
- Always review data before submitting or saving changes
- Use search and filter options to quickly find specific records
- Check for validation errors and correct them before saving
- Log out when you are done to protect your account security

---

## 💡 Tips & Best Practices

1. Review your profile periodically to ensure all information is current and accurate.
2. Upload all required documents during onboarding to complete your employee file.
3. Update emergency contact information whenever there are changes.
4. Keep your bank account details updated for accurate salary processing.
5. Check your employment details if you notice any discrepancies in your records.
