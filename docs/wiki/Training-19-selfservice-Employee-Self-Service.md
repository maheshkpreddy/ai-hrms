## 📋 Module Overview

The Employee Self-Service module empowers employees to manage their own HR-related tasks without depending on HR staff. Employees can view and update their profiles, apply for leaves, access payslips, and raise service requests. The module reduces HR workload by enabling employees to handle routine tasks independently while maintaining proper approval workflows for sensitive actions. It provides a convenient one-stop portal for all employee HR needs.

The Employee Self-Service module is accessible from both the sidebar navigation and the module home page grid. It contains the following sub-items that provide specialized functionality within the module.

---

## 🔧 Sub-Items & Features

### 1. My Profile

View and update personal information including contact details, address, emergency contacts, and bank account information. Changes to sensitive fields may require HR approval.

**Steps to use My Profile:**
1. Navigate to the Employee Self-Service module from the sidebar or module home page
2. Click on "My Profile" from the sub-menu
3. Use the available options to create, view, edit, or manage records

## 📝 Step-by-Step Usage Instructions

### Getting Started
1. **Log in** to eh2r AI at [https://ai-hrms-rho.vercel.app](https://ai-hrms-rho.vercel.app) with your credentials
2. **Navigate** to the Employee Self-Service module using the sidebar or module home page
3. **Select** the appropriate sub-item from the module menu
4. **Perform** the desired action (create, view, edit, or manage records)
5. **Save** your changes and verify the results

### Common Actions
- **My Profile**: Access via Employee Self-Service → My Profile
- **My Leaves**: Access via Employee Self-Service → My Leaves
- **My Payslips**: Access via Employee Self-Service → My Payslips
- **Raise Request**: Access via Employee Self-Service → Raise Request

### Best Practices for Employee Self-Service
- Always review data before submitting or saving changes
- Use search and filter options to quickly find specific records
- Check for validation errors and correct them before saving
- Log out when you are done to protect your account security

## 💡 Tips & Best Practices

1. Keep your profile information updated for accurate records and communication.
2. Apply for leaves in advance whenever possible to help managers plan team capacity.
3. Download and save payslips for tax filing and financial planning purposes.
4. Use the raise request feature for any workplace needs instead of informal channels.
5. Check your leave balance before applying to avoid rejections due to insufficient balance.
